# Cortex Planning Surface

This directory holds non-canonical planning, prompt, and implementation-shaping artifacts.

It is not the constitutional source of repo truth.

Canonical repo truth remains in:

- top-level constitutional files such as `PROJECT_CHARTER.md`, `LOCAL_DOCTRINE.md`, and `AUTHORITY_BOUNDARIES.md`
- admitted or draft lane contracts under `docs/contracts/`
- governed source-lane materials under `docs/source-lanes/`
- ADRs under `DECISIONS/`
- canonical Scrivener fixture evidence under `fixtures/scrivener/`

Use this directory for:

- historical implementation plans
- bounded build-preparation notes
- governance-preparatory prompt packets
- special-track planning notes that are explicitly not admission or implementation authorization

Do not treat files here as:

- admitted contracts
- runtime requirements by themselves
- evidence of implementation authorization
- replacements for ADRs or canonical contracts
